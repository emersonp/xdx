# xdx

A collection of simple dice rolling functions for Clojure.

##Usage
### Examples
Roll 3d6.

    (roll-dice 3 6)

Roll 4d8.

    (roll-dice 4 8)

Roll 2d10, exploding (*a la Savage Worlds*).

    (role-explode 2 10)

## Options

None, currently.
