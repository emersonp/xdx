# xdx

A collection of simple dice rolling functions for Clojure.

##Usage
### Examples
Roll 3d6.

    (roll "3d6")
    (roll :3d6)

Roll 4d8.

    (roll "4d8")
    (roll :4d8)

Roll 2d10, exploding (*a la Savage Worlds*).

    (roll "2x10")
    (roll :2x10)

Roll 2 Fudge dice (*a la FATE*).

    (roll "2f")
    (roll :2f)

## Options

None, currently.
